# OAuth 2 Demo

This is based on the demo provided at https://www.oauth.com/oauth2-servers/accessing-data/

I made some small changes to make it work.

## Use

create .secrets.php

edit .secrets.php:

    <?php
    // Fill these out with the values from Github
    $githubClientID = '';
    $githubClientSecret = '';
    ?>

run `php -S 127.0.0.1:8000`
